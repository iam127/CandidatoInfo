# CandidatoInfo - Transparencia Electoral Ciudadana

## 📱 Descripción
Aplicación móvil que permite a los ciudadanos consultar información pública sobre candidatos al Congreso y Presidencia del Perú, incluyendo denuncias, proyectos presentados, historial político y enlaces a fuentes oficiales.
